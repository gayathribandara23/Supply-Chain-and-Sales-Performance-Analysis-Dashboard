# Supply Chain and Sales Performance Analysis Dashboard

## Project Objective

The objective of this project is to develop an interactive Power BI dashboard that analyzes sales performance, profitability, product performance, customer contributions, and regional performance using business data. The dashboard helps transform raw data into meaningful insights, enabling users to monitor key business metrics, identify trends, evaluate performance across products and regions, and support data-driven decision-making for improved business efficiency and growth.
